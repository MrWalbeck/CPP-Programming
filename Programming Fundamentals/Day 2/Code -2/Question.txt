Write a C++ program that:
1. Takes two integers as input from the user
2. Compares the two numbers and displays the relationship between them
3. Output should indicate if the numbers are:
    - Equal to each other
    - One is greater than the other
    - One is less than the other
    - Not equal to each other

Example Output:
- If num1 = 5 and num2 = 5: "5 is equal to 5"
- If num1 = 10 and num2 = 3: "10 is greater than 3"
- If num1 = 2 and num2 = 8: "2 is less than 8"
- If num1 = 7 and num2 = 4: "7 is not equal to 4"
