# 🚀 C++ From Scratch - University Students

Welcome to a complete and structured C++ learning journey designed specifically for university students and beginners.

This repository documents step-by-step progress in mastering C++ — starting from basic syntax and moving towards advanced concepts like Object-Oriented Programming, Standard Template Library (STL), and Data Structures & Algorithms.

---

## 🎯 Project Mission

The goal of this repository is to:

- Build strong C++ fundamentals  
- Practice coding consistently  
- Prepare for university exams  
- Strengthen problem-solving skills  
- Prepare for technical interviews  

This is not just a cheatsheet — it is a complete learning path.
